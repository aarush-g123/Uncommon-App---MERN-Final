# Project Overview - UncommonApp: College Application Tracker

Keeping track of college applications can be difficult.

With UncommonApp, Students track college applications in spreadsheets. They will be able to browse a list of colleges and add/remove/update applications. 

## Planned Features
- User accounts
- Sign in with Google OAuth
- Add colleges
- Delete Colleges
- Edit application status (Not Started, Applied, Accepted, Rejected, EA, RD, ED)
- Sort/filter schools
- Search and filter applications by status
- Notes section for each college
  - Level of interest
  - Number of essays
  - SAT/ACT
  - Application fees

## Planned Technologies/Libraries
- Google OAuth
- Sequelize
- Postgres
- Node.js 
- React
- Express.js

# Project Structure
The frontend is in the "frontend" folder, and the backend is in the "backend" folder.
